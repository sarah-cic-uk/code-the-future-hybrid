AI Session - Lesson 4

Models, tokens and costs

Different AI tools charge differently, think differently, and handle code in different ways. The safest choice is not always the biggest or most expensive model.

Think of models like different tools in a toolbox. Some are fast and cheap, some are smarter but cost more, and some are better for code or longer explanations.

Understanding Key Concepts

Models

Small / Fast Models
Good for simple tasks and quick checks
Larger Models
Better for tricky reasoning or longer code
Pricing Tiers
Free tiers, paid tiers, or usage limits

Tokens

Tokens are chunks of text the AI reads and writes. More text in your prompt or pasted code usually means more tokens, which can mean more cost.

Important: Long conversations also use tokens because the model has to remember the earlier messages.

Cost Surprises

  • Big pasted files use lots of tokens quickly
  • Long answers cost more than short ones
  • Several retries in a row can add up

Model Comparison

Model Type Best For Cost Speed
Small Models Quick questions, simple explanations, basic code completion Low / Free Fast
Medium Models Code review, debugging, moderate complexity tasks Moderate Medium
Large Models Complex reasoning, architecture decisions, detailed analysis Higher Slower

How to Avoid Unexpected Costs

Best Practices

  • Use the smallest model that does the job
  • Keep prompts focused and short when you can
  • Paste only the code you need help with
  • Ask for a brief answer first, then ask for more if needed
  • Check pricing, usage limits, and free tiers before you start

Safe Workflow

  1. Start with a small task
  2. Use a low-cost or free option first
  3. Check the answer before asking again
  4. Only switch to a bigger model if the problem is actually hard

Professional Mindset

Best practice

If you are paying for AI, treat it like any other tool with a budget: plan it, watch it, and stop it from running away on you.

Interview-ready line

"I choose the cheapest tool that can do the job, keep my prompts tight, and watch token usage so I avoid surprise bills."

Remember

The best AI setup is the one that helps you learn, build, and stay in control of the cost.
